# 🐲 Dragon.io

A Slither.io-style multiplayer snake game built with Node.js, Express, and Socket.IO.

## 🚀 Déploiement Gratuit

### Railway (Recommandé)
[![Deploy on Railway](https://railway.app/button.svg)](https://railway.app/new/template?template=https%3A%2F%2Fgithub.com%2FMonkeyDNikaa%2Fdragon-io-game&envs=NODE_ENV%2CPORT&NODE_ENVDefault=production&PORTDefault=3000)

**Clique le bouton ci-dessus** → Connecte avec GitHub → Déploiement automatique

### Cyclic.sh (Alternative sans carte)
1. Va sur https://cyclic.sh
2. Sign in with GitHub
3. Deploy → Select `dragon-io-game`
4. Fait !

### Glitch (Instantané)
[![Remix on Glitch](https://cdn.glitch.com/2703baf2-b643-4da7-ab91-7ee545c74f1b%2Fremix-button.svg)](https://glitch.com/edit/#!/import/github/MonkeyDNikaa/dragon-io-game)

## 🎮 Features

- Real-time multiplayer
- 5 AI bots
- 6000x6000 world
- 800 food items
- Portal teleportation
- Mobile & desktop support

## 🛠️ Local Development

```bash
npm install
npm start
# Open http://localhost:3002
```
